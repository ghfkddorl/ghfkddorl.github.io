    HTML MD 파일은 리소스로 사용하기

    Apr 12, 2019
    2 min read

    휴고는 렌더링시 페이지 번들 안에 있는 인덱스 파일은 제회한 다른 html 파일과 md 파일을 렌더링 해주지 않습니다. 페이지 번들 안에 있는 다른 파일들은 렌더링된 페이지파일과 같은 폴더 안으로 복사되지만 html 파일과 md 파일은 그렇지 않습니다. 버그인지 원래 그런 기능인지는 모르겠지만, 주피터 노트북이나 plotly를 사용하기 위해서 이런 기능이 필요합니다. 따라서 나름 알아본 방법을 적어봅니다.

    1. 숏코드 이용

    우선 숏코드를 이용해 마크다운이나 html 파일을 페이지 파일에 병합하는 방법이 있습니다. 숏코드는 렌더링 되기전의 페이지 파일에 원하는 내용을 넣어줍니다. 따라서 마크다운과 html 파일의 내용을 페이지 파일안에 넣어주는 숏코드를 만듭니다.는 html은 병합하는 숏코드는 아래와 같습니다. 숏코드를정의한 파일의 이름은 숏코드의 이름과 같아야하며, 위치는 당연히 layout/shortcode 파일 안에 있어야 합니다.

    아래의 경우 파일이름은 merge_html.html 이며 파일안에 숏코드의 이름은 merge_html 입니다. 이름에 _가 들어가 있는 경우 마크다운에서 _를 특수문자로 사용하기 때문에, 일반글자로 사용하기위해선 \_를 입력해야 합니다. 숏코드로 만들어 지는 태그의 클래스를 short로 지정해준 이유는 MathJax 라이브러리를 이용할 경우 해당 클래스를 MathJax에 예외로 지정해주어야 하기 때문입니다.
